# 🛠️ System Technologies & Skills (Ramaniya)

This file documents the massive array of technical skills, frontend architectures, and system algorithms utilized natively to construct **Ramaniya**.

## Frontend Technologies
- **React 18**: Core Component Architecture and Hooks (useState, useEffect) memory management.
- **Vite (Rollup)**: Heavy development engine prioritizing Hot-Module-Replacement (HMR) speeds.
- **Tailwind CSS 3**: Completely utility-first styled elements avoiding CSS bloat.
- **React Router DOM v6**: Protected client-side routing, query parsing, and seamless linear progression layouts.

## Data Visualization & UI Mathematics
- **Recharts (D3.js SVG Portals)**: Dynamically plotting massive chronological arrays (historical NAVs) into smooth mathematical vector representations.
- **Lucide-React**: Elite SVG-based iconography seamlessly matching typography weights.

## API Integration & Telemetry Networks
- **MFAPI (Live)**: Custom `fetch()` parsing logic routing asynchronous mutual fund datasets, stripping API payloads, and reformatting them into client models.
- **Marketaux API**: Direct temporal-ranged query parameters executing real API pulls equipped with mathematical fallbacks preventing total system UI failures.

## System Design & Browser Storage
- **Simulated Payment Gateway**: Custom abstraction mimicking Razorpay's asynchronous token transactions.
- **Client-Side Storage Layers**: Utilizing browser `LocalStorage` specifically formatted in JSON structures mapping persistent session IDs, KYC completion flags, and dynamic transactional hashes.
- **Graceful Failovers**: Comprehensive fallback arrays kicking in natively during local HTTP fetch failures so the design continues performing immaculately.
